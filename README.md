# 局域网串联方案总览

两台 Windows 电脑在同一路由器下，实现一套键鼠、互传文件、共享音频、远程 AI 操控。

---

## 硬件

```
路由器 ImmortalWrt (192.168.10.1, OpenWrt, WiFi 6)
├── benben (台式机, 192.168.10.231, 插键鼠)
├── xiaohaifale (笔记本, 192.168.10.121)
└── realme-GT-7 (手机, 192.168.10.107)

WiFi: fale520 / 5G+2.4G
```

---

## 能力矩阵

| 需求 | 工具 | 触发方式 |
|------|------|----------|
| 一套键鼠控两台 | Deskflow | 台式机开机自启 |
| 局域网传文件 | LocalSend | 台式机开机自启 |
| 音频互通 | SonoBus | 台式机开机自启 |
| 后台远控笔记本 | SSH | 免密，随时可用 |
| AI 视觉操控笔记本 | helix-pilot + Ollama | 笔记本连 fale520 自动启动 |
| 全家网络代理 | 路由器 | 常驻 |
