## 笔记本远控

### SSH（✅ 已配置，免密 + 自启）

直接连，无需密码：

```powershell
ssh XIAOHAI@192.168.10.121
```

**踩坑备忘**：
- 用户 `xiaohai` 的实际 profile 目录是 `C:\Users\Administrator`（不是 `C:\Users\XIAOHAI`）
- `authorized_keys` 权限必须只有 `SYSTEM` 和用户本人，不能有 `Administrators` 组
- 私钥不能有 passphrase 否则 `BatchMode` 会挂

<details>
<summary>首次安装命令（笔记本管理员 PowerShell，仅一次）</summary>

```powershell
Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0
Set-Service sshd -StartupType Automatic
Start-Service sshd
New-NetFirewallRule -Name OpenSSH -DisplayName OpenSSH -Enabled True -Direction Inbound -Protocol TCP -Action Allow -LocalPort 22
```
</details>

## 路由器操作备忘

### SSH 进去
```python
import paramiko
s = paramiko.SSHClient()
s.set_missing_host_key_policy(paramiko.AutoAddPolicy())
s.connect('192.168.10.1', username='root', password='<密码>')
si, so, se = s.exec_command('iwinfo ra0 info')
print(so.read().decode())
```

### 常用命令
```bash
uci show wireless              # 查看无线配置
iwinfo ra0 info                # 2.4G 状态
iwinfo rax0 info               # 5G 状态
cat /tmp/dhcp.leases           # DHCP 分配列表
wifi reload                    # 重启 WiFi
```
