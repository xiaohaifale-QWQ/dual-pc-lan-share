# 局域网键鼠共享 + 文件传输 + 音频互联 方案

---

## 🖥️ 适用场景

两台或多台 Windows 电脑，在同一局域网（或 Tailscale 虚拟网络）下，**共用一套键盘鼠标**、**互传文件**、**共享音频**。

---

## 📦 需要的软件

| 软件 | 用途 | 版本 | 大小 |
|------|------|------|------|
| **Deskflow** | 局域网键鼠共享 | v1.26.0 便携版 | ~46MB |
| **LocalSend** | 局域网文件传输 | v1.17.0 | ~18MB |
| **SonoBus** | 局域网音频流 | v1.7.2 | ~81MB |

全部开源免费，全在 `E:\KVM\` 目录下。

## 🔧 安装步骤

### 下载链接

- Deskflow: `https://github.com/deskflow/deskflow/releases/latest` → `win-x64-portable.7z`
- LocalSend: `https://github.com/localsend/localsend/releases/latest` → `windows-x86-64.zip`
- SonoBus: `https://github.com/sonosaurus/sonobus/releases/latest` → `win.exe`

### 安装命令

```powershell
# Deskflow（用 7-Zip 解压便携版到 E:\KVM）
7z x deskflow-1.26.0-win-x64-portable.7z -oE:\KVM

# LocalSend（解压到 E:\KVM\LocalSend）
Expand-Archive LocalSend-1.17.0-windows-x86-64.zip -DestinationPath E:\KVM\LocalSend

# SonoBus（静默安装到 E:\KVM\SonoBus）
sonobus-1.7.2-win.exe /VERYSILENT /DIR=E:\KVM\SonoBus
```

### 创建快捷方式

```powershell
$WshShell = New-Object -ComObject WScript.Shell

$s = $WshShell.CreateShortcut("E:\KVM\Deskflow.lnk")
$s.TargetPath = "E:\KVM\deskflow-1.26.0-win-x64-portable\deskflow.exe"
$s.WorkingDirectory = "E:\KVM\deskflow-1.26.0-win-x64-portable"
$s.Save()

$s = $WshShell.CreateShortcut("E:\KVM\LocalSend.lnk")
$s.TargetPath = "E:\KVM\LocalSend\localsend_app.exe"
$s.WorkingDirectory = "E:\KVM\LocalSend"
$s.Save()

$s = $WshShell.CreateShortcut("E:\KVM\SonoBus.lnk")
$s.TargetPath = "E:\KVM\SonoBus\SonoBus.exe"
$s.WorkingDirectory = "E:\KVM\SonoBus"
$s.Save()
```

## ⚙️ Deskflow 配置（键鼠共享）

### 概念

| 角色 | 说明 |
|------|------|
| **Server** | 插键鼠的那台电脑 |
| **Client** | 被控制的其他电脑 |

### Server 端设置

1. 打开 Deskflow → 选 **Server** 模式
2. 点 **Configure Server** → 拖一个电脑图标到格子（放右边）
3. 双击图标 → Screen name 填 Client 的名字（大小写一致）
4. 关闭 TLS：**设置 → 取消勾选 "启用 TLS 加密"**
5. 点 **Start**

### Client 端设置

1. 打开 Deskflow → 选 **Client** 模式
2. **Server IP** 填 Server 的局域网 IP（如 `192.168.10.231`）
3. 关闭 TLS：**设置 → 取消勾选 "启用 TLS 加密"**
4. 点 **Start**

### 快捷键

- **Scroll Lock** → 锁定/解锁鼠标跨屏（一键开关）
- 自定义热键 → Server 设置 → Hotkeys

### 排坑指南

| 问题 | 原因 | 解决 |
|------|------|------|
| TLS 证书错误 `failed to generate pem` | 中文路径/KVM 路径写不进去 | 两台都关 TLS |
| Client 连不上 `Timed out` | 防火墙挡了 | 管理员 PowerShell 加规则 |
| `cannot bind address [10049]` | Deskflow 绑了已断开的网卡 IP | 退出 Tailscale → 启动 Deskflow → 再开 Tailscale |
| LAN 互相 ping 不通 | 路由器 AP 隔离 | OpenWrt: 网络→无线→编辑→取消"隔离客户端" |

### 防火墙命令（管理员 PowerShell）

```powershell
netsh advfirewall firewall add rule name="Deskflow TCP" dir=in action=allow protocol=TCP program="E:\KVM\deskflow-1.26.0-win-x64-portable\deskflow-core.exe" enable=yes
```

## 📎 LocalSend 用法（文件传输）

1. 两台都打开 LocalSend
2. 同一局域网自动发现对方
3. 拖文件 → 点对方设备名 → 对方点接收
4. 不需要互联网，HTTPS 加密传输

---

## 🎵 SonoBus 用法（音频共享）

1. 两台都打开 SonoBus
2. **Group** 输入同一个房间名（如 `benben-room`）
3. 点 **Connect**
4. 选输入/输出设备 → 声音互通
5. 只要**听**另一台声音 → 把本机 M 键 Mute 掉
