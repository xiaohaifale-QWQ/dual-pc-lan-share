# 路由器 OpenClash 排障备忘

## 代理服务

路由器已运行代理，局域网设备可通过代理访问外网。

> 具体配置略。

## 代理连接问题

**现象**：部分网站/服务（如 ollama.com）无法访问。

**根因**：代理的 DNS 劫持模式对未在规则中的域名返回假 IP，导致直连失败。

**解决**：在代理的假 IP 过滤列表中添加目标域名。

## 常用命令

```bash
cat /tmp/dhcp.leases           # DHCP 分配列表
wifi reload                    # 重启 WiFi
```
